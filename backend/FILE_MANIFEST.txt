DATABASE TABLES MAPPED
================================================================================

1. pessoas (People/Individuals)
   JPA Entity: Pessoa.java
   Columns: id, nome, telefone, email, endereco, cidade, codigo_postal, nif

2. funcionarios (Employees)
   JPA Entity: Funcionario.java
   Columns: id, pessoa_id, posicao, data_admissao, data_saida, salario, ativo

3. utilizadores (Users/Accounts)
   JPA Entity: Utilizador.java
   Columns: id, username, password, pessoa_id, role, ativo, data_criacao, ultimo_acesso

4. vinhos (Wines/Products)
   JPA Entity: Vinho.java
   Columns: id, nome, tipo, descricao, regiao, ano_colheita, preco, stock, stock_minimo, ativo, data_criacao

5. vendas (Sales)
   JPA Entity: Venda.java
   Columns: id, utilizador_id, cliente_id, data_venda, total, status

6. itens_venda (Sale Items)
   JPA Entity: ItemVenda.java
   Columns: id, venda_id, vinho_id, quantidade, preco_unitario, subtotal

HOW TO USE THIS PROJECT
================================================================================

1. SETUP (10-15 minutes)
   - Run: ./QUICK_START.sh
   - Or follow: SETUP_GUIDE.md

2. BUILD
   - mvn clean install

3. RUN (Development)
   - mvn spring-boot:run -Dspring-boot.run.arguments="--spring.profiles.active=dev"

4. RUN (Production)
   - java -jar target/vinhadouro-backend-1.0.0.jar

5. TEST
   - curl http://localhost:8080/api/health
   - curl -X POST http://localhost:8080/api/login ...

6. INTEGRATE
   - Connect frontend to API endpoints
   - Use JSON request/response bodies
   - Handle error codes appropriately
